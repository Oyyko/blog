"latex-workshop.latex.tools": [
        {
            "name": "latexmk",
            "command": "latexmk",
            "args": [
                "-synctex=1",
                "-interaction=nonstopmode",
                "-file-line-error",
                "-pdf",
                "-outdir=%OUTDIR%",
                "%DOC%",
                "-xelatex"
            ],
            "env": {}
        },

即可

即在命令参数的最后增加 -xelatex 参数

备注：之前在文件头部添加% !TEX program = xelatex是管用的，现在不知道为啥又不行了。
