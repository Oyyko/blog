#include <iostream>
using namespace std;
void f(double x)
{
    cout << "DOUBLE" << endl;
}

void f(int &x)
{
    cout << "INT&" << endl;
}

int main()
{
    f(2);
}

这段代码会输出DOUBLE 原因在于 2是右值 不能用于初始化一个int& 则只能选择第一个版本的函数重载
